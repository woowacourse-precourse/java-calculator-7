- [x] 기본(쉼표, 콜론) 구분자로 각 숫자의 합을 반환하는 기능

- [ ] 커스텀 구분자로 각 숫자의 합을 반환하는 기능
- [ ] 예외 처리

- [ ] 테스트

    - [ ] 커스텀 구분자 테스트
