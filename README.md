## 💻 문자열 덧셈 계산기 기능 목록

### 🛠 구현할 기능

- [ ] 문자열 입력 기능
  - [ ] 사용자 문자열 입력 기능
  - [ ] 구분자와 양수로 구성된 문자열인지 검증
  - [ ] 커스텀 구분자가 올바르게 입력되었는지 검증

- [ ] 문자열 숫자 추출 기능
  - [ ] 기본 구분자(쉼표, 콜론)를 기준으로 숫자 추출
  - [ ] 커스텀 구분자를 기준으로 숫자 추출

- [ ] 결과 출력 기능
  - [ ] 추출된 숫자 덧셈
  - [ ] "결과 : n" 형식으로 출력
