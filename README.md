# java-calculator-precourse

## 구현할 기능 목록
**1. 구분자를 찾는 기능**
- 기본 구분자(쉼표, 클론)을 찾는다
- 커스텀 구분자 (“//“와 “\n” 사이)를 찾는다.

**2. 숫자를 추출하는 기능**
- 문자열로부터 숫자를 추출한다.
- 문자열이 빈 문자열인 경우 0을 추출한다.

**3. 계산하는 기능**
- 숫자들을 더한다.

**4. 잘못된 입력에 대한 예외 처리 기능**
- 숫자가 음수인 경우 예외를 처리한다.
- 그 외 잘못된 입력에 대한 예외를 처리한다.(추가 예정)

**5. 입출력 처리 기능**
- 문자열을 입력받는다.
- 결과값을 출력한다.
