# java-calculator-precourse
___

## InputHandler
> 입력을 처리하는 클래스
### Reader
> 입력을 수행하는 클래스


## OuputHandler
> 출력을 처리하는 클래스
### Printer
> 출력을 수행하는 클래스


## NumberHandler
> 숫자를 관리하는 클래스
### Number
> 숫자를 구체화 한 클래스


## DelimiterHandler
> 구분자를 관리하는 클래스
### Delimiter
> 구분자를 구체화 한 클래스

## UserInputParser
> 사용자 입력을 파싱하는 클래스

## ServiceHandler
> 도메인 흐름을 담당하는 클래스

## Controller
> 입출력을 담당하고 ServiceHandler에게 도메인 로직을 요청하는 클래스
