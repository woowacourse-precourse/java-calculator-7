# 문자열 덧셈 계산기

입력한 문자열에서 숫자를 추출하여 더하는 계산기

## 구현 기능 목록

1. 입력 처리

- [X] 입력 프롬프트 출력
- [X] 사용자 입력 기능

2. 입력 문자열 검증 및 구분

- [ ] 커스텀 구분자 처리
    - [ ] 문자열 맨 앞의 `//`와 `\n` 사이에 위치한 구분자 추출
- [ ] 문자열 분리 및 숫자 추출
    - [ ] 기본 구분자(쉼표`,`, 클론`:`)로 구분
    - [ ] 커스텀 구분자가 있을 경우 이를 기준으로 문자열 분리
    - [ ] 추출된 각 요소가 숫자(양수)인지 확인하고, 숫자로 변환
- [ ] 예외 처리: 잘못된 값(구분자 또는 숫자 이외의 값)이 포함된 경우 `IllegalArgumentException` 발생
    - [ ] 빈 문자열이 입력된 경우

3. 덧셈 계산

- [ ] 숫자들의 합을 계산

4. 결과 출력

- [ ] 결과 프롬프트 출력
- [ ] 결과 값 출력

5. 애플리케이션 종료

- [ ] 정상 종료: 결과 출력 후 애플리케션을 정상 종료
- [ ] 예외 발생 시 종료: 예외가 발생할 경우 오류 메시지를 출력하고 애플리케이션을 종료
